sound

Other than giving user feedback visually, we can also provide auditory feedback.

To add some sounds, we use the Pygame.mixer.Sound()

# play.py
    def on_init(self):

        self.type_sound = pygame.mixer.Sound("typewriter-snippet-02.wav")

And then play the sound when we have a keyhit:

    def check_keyhit(self):
        if not self.keyhit:
            return

        self.type_sound.play()

Feel free to add more sounds of your liking:

    self.type_sound = pygame.mixer.Sound("typewriter-snippet-02.wav")

    self.err_sound = pygame.mixer.Sound("glitch.wav")
    self.success_sound = pygame.mixer.Sound("typewriter-ding-near-mono.wav")
