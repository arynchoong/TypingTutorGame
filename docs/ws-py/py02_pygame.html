PyGame

Pygame is a python library that is built on top of the SDL (Simple DirectMedia Layer) library that allows for developers to have low level access to common gaming tools of audio, peripherals and graphics.

In short it is a cross platform tool to easily develop games in Python.

Install

The PyGame version 2.0 is currently still in dev as at 1st October 2019, and as such we need to uninstall any older version and build the dev version from source.

There are crucial updates in version 2.0 that we will need for our Typing Tutor Game. Namely the key reads.

Please see: https://www.pygame.org/wiki/GettingStarted#Installing%20From%20Source

and install PyGame Dev version 2.0.0.DEV from https://github.com/pygame/pygame.git

Test your installation by calling the example that comes with the package:

python3 -m pygame.examples.aliens

PyGame Basics

PyGame is not compatible with jupyter notebooks.

So we will use scripts. Let's start with a script and let's name it 'game.py'.

To use pygame, lets import the package and it's keywords.

import pygame
from pygame.locals import *

Now fill in the basics of a python script, and we need to initalise the pygame module to use it:

def main():
    pygame.init()
    
if __name__=="__main__":
    main()

At the moment, we haven't done much.

So let us create out game screen, and call it our display surface:

def main():
    pygame.init()
    pygame.display.set_caption('Typing Tutor Game')
    disp_surf = pygame.display.set_mode((640, 400))

set_caption() sets the game caption (application title) on the top of the display.

The display surface is our game's screen, or game play surface area, where our game runs.

Game loop

And any game will require user input in some form or another, let us now add the codes to check for trigger to close the game. (user clicking the red cross icon on top corner of our display surface)

    disp_surf = pygame.display.set_mode((640, 400))
    running = True
    
    while running:
        for event in pygame.event.get():
            if event.type == pygame.QUIT:
                running = False

The codes set a flag called running and check all the events for the game, if it gets the event pygame.QUIT, it exits the loop.

Test your codes by calling the script:

python3 -m game.py

And hit the red cross button to quit.
