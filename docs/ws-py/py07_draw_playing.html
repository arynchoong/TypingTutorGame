Draw Game Play

We want have a blue background, with light blue city line, white words, with characters that turns green when typed. Animate a red arrow destroying the word when typed.

Let's prepare our colours for the screen: 

# play.py

#               R    G    B
BLUE        = (  0,   0, 128)
LIGHTBLUE   = (135, 206, 235)
WHITE       = (255, 255, 255)
GREEN       = (  0, 200,   0)

Draw our playing screen by filling it in with blue colour as background: 

    def on_render(self):
        self._display_surf.fill(BLUE)

Draw words

To draw the words, we need to draw the text for each word in game play.

Let's prepare the font for the game play: 

class GamePlay():
    def __init__(self, display_surf):

        self.font = pygame.font.Font('Consolas Bold.ttf', 16)

Similarly to how we draw our main texts, let's draw our words in our game play.py: 

    def draw_word(self, text, colour, coord):
        surf = self.font.render(text, True, colour)
        rect = surf.get_rect()
        rect.topleft = (x, y)
        self._display_surf.blit(surf, rect)
        return

Now we can easily call the method to draw our words onto the display screen and then update: 

    def on_render(self):
        self._display_surf.fill(BLUE)

        # words
        for word in self.game_objects.game_words:
            self.draw_word(word.text, GREEN, word.coord())
        pygame.display.flip()
